# Migrate your communities from one account to another

## Usage
```
usage: lemmy_migrate [-h] -s source_site -a source_account -p source_password -d destination_site -l destination_account -c
                     destination_password

Migrate subscribed communities from one account to another

options:
  -h, --help            show this help message and exit
  -s source_site        Source site URL
  -a source_account     Source account
  -p source_password    Source password
  -d destination_site   Destination site URL
  -l destination_account
                        Destination account
  -c destination_password
                        Destination password
```
